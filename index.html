CONCEITO

Finanças App é um Sistema Financeiro de forma a permitir que qualquer pessoa possa ter um sistema de educação financeira e controle de finanças pessoais ou para pequenas empresas. A ideia do sistema é permitir que qualquer pessoa possa ter um sistema de educação financeira e controle de finanças pessoais, apesar de ser útil para pequenas empresas, pois é mono-usuário, ou seja, cada usuário possui suas próprias contas e lançamentos.

PREMISSA

A premissa deste projeto é garantir que possa ter um sistema a executar onde sentir-se mais seguro.

TECNOLOGIA

O software foi desenvolvido em Typecript a usar NestJs e TypeOrm para o backend e Angular para o frontend e banco de dados MySQL Community Edition 8.0.37.
