API-CONSOLIDADO

Definição

A API-CONSOLIDADO é um serviço REST API destinado a obtenção do consolidado das finanças.

Tecnologia

Foi concebido em Typescript com NestJs de forma a permitir escalar (verticalmente e/ou horizontalmente).

Documentação

A documentação do código é gerada pela biblioteca compodoc e deve ser executada sempre que houver mudanças no software.

Para executar, usar:

Example :
   > npm run compodoc

É criada a pasta "documentation" na pasta raiz do projeto e deve ser executada pelo arquivo index.html.

Configuração

Desenvolvimento

Copiar o arquivo ".env.sample" para ".env"

Produção

Enviar os parâmetros para o docker.

Execução

O software deve ser executado com o orquestrador a partir de seu arquivo main.ts ou com o comando interno para desenvolvimento:

Example :
   > npm run start:dev

Contribuição

Este projeto tem como objetivo criar um software opensource para administração de lançamentos financeiros simples de forma a democratizar a utilização de um gestor de finanças. O projeto é criado e mantido por Andre Mesquita a aproveitar minha paixão de criar soluções e desenvolver software. Se encontrar valor em algo, ficaria grato por sua contribuição para o suporte deste aplicativo e, neste caso, patrocinar-me.
